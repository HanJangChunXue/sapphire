Enhancements in Sapphire 0.7.2

  1. Expression Language
    1. Use EL in Wizard Label
    2. Use EL in Wizard Page Label and Description

Use EL in Wizard Label

Use the expression language when defining the wizard label (window title).

<wizard>
    <label>new ${ Type } file</label>
</wizard>

Use EL in Wizard Page Label and Description

Sapphire Wizard Page's label and description are now using expressions and their content can be controlled dynamically.

Example

<wizard>
    <page>Occupation       
       <label>${ Spouse.Name.First }'s occupation</label>
       <description>${ Spouse.Name.First }'s job description</description>
       ...
    </page>
</wizard>

Similiarly, Sapphire Wizard's label and description are also using expressions and their content can be controlled dynamically.

Example

<wizard>
    <label>${ Contact.Name }'s task</label>
    <description>${ Contact.Name }'s todo list</description>
    ...
</wizard>
